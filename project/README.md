# Citizen AI - Community Engagement Platform

A modern web application that empowers communities through intelligent analysis, sentiment monitoring, and AI-driven insights for citizen engagement and local government services.

## 🚀 Features

- **AI-Powered Chat Assistant**: Intelligent chatbot for 24/7 citizen support
- **Sentiment Analysis**: Real-time analysis of community sentiment and concerns
- **Concern Reporting**: Digital platform for citizens to report issues with automatic categorization
- **Analytics Dashboard**: Comprehensive data visualization and insights
- **Feedback System**: Structured feedback collection and processing

## 🛠️ Technology Stack

### Frontend
- **React 18** with TypeScript
- **Vite** for fast development and optimized builds
- **Tailwind CSS** for modern, responsive styling
- **React Router DOM** for client-side routing
- **Recharts** for data visualization
- **Lucide React** for icons

### AI Integration
- **IBM Granite 3.3-2B Instruct Model** (demonstration code)
- **Hugging Face Transformers** integration
- **Custom sentiment analysis** and text classification
- **Multi-language support** for diverse communities

- **Natural Language Understanding**: Processing citizen queries and concerns

## 🔧 Development

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation
```bash
# Clone the repository
git clone <repository-url>
cd citizen-ai

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production
```bash
npm run build
```
